# xhs-smart-companion - 测试指南

## 项目状态
✅ 所有文件已创建，项目结构完整

## 本地测试步骤

### 1. 在 Chrome 中加载扩展
1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 点击右上角「开发者模式」开关
4. 点击「加载已解压的扩展程序」
5. 选择目录：`/Users/chenxinyu/Desktop/chajian/xhs-smart-companion`

### 2. 测试功能

#### 测试宠物组件
1. 打开小红书或抖音
2. 等待页面加载完成
3. 观察右下角是否出现宠物图标
4. 点击宠物，查看迷你面板是否显示
5. 查看宠物是否显示欢迎提示

#### 测试提取功能
1. 访问小红书收藏页面或抖音页面
2. 点击宠物 →「提取当前页内容」
3. 观察宠物状态变化（idle → thinking → happy）
4. 查看提示信息是否显示保存数量

#### 测试知识库
1. 点击宠物 →「打开知识库」
2. 或从浏览器工具栏点击插件图标 → 侧边栏
3. 查看收藏统计数据
4. 测试「导出 Markdown」功能

#### 测试弹出窗口
1. 点击浏览器工具栏的插件图标
2. 查看弹出窗口是否正常
3. 测试按钮功能

### 3. 控制台调试
如果遇到问题，在页面打开 DevTools（F12）查看 Console 和 Extensions 错误页面。

## 注意事项
- 本插件仅在 `xiaohongshu.com` 和 `douyin.com` 网站上运行
- 依赖 Chrome Manifest V3，需要 Chrome 88+ 版本
- 不提供图标文件（可选），但扩展仍可正常工作
