# xhs-smart-collection - 小红书智能收藏夹助手

一个 Chrome 浏览器插件，帮助你提取、管理和导出小红书收藏内容。

## 功能特性

1. **提取收藏** - 从当前小红书页面提取可见的收藏卡片信息
2. **智能标签** - 基于本地关键词规则自动生成标签
3. **搜索过滤** - 支持按标题、作者搜索，按标签过滤
4. **Markdown 导出** - 导出收藏内容为 Markdown 格式，适合复制到 Obsidian

## 技术实现

- **Manifest V3** - 使用最新的 Chrome 扩展规范
- **原生 JavaScript** - 不依赖任何框架
- **chrome.storage.local** - 本地存储收藏数据
- **content script** - 从页面提取收藏信息

## 本地测试步骤

### 1. 准备图标文件

由于图标文件需要是真正的 PNG 格式，你需要替换 `icons/` 目录中的图标文件：

- 创建 16x16、48x48、128x128 像素的 PNG 图标
- 替换 `icons/icon16.png`、`icons/icon48.png`、`icons/icon128.png`

### 2. 加载扩展到 Chrome

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择 `xhs-smart-collection` 目录

### 3. 测试功能

1. 打开小红书网站并登录
2. 访问你的收藏页面：`https://www.xiaohongshu.com/favorites`
3. 确保页面上有可见的收藏卡片
4. 点击浏览器工具栏中的插件图标
5. 点击「提取当前页面收藏」按钮
6. 查看提取的收藏内容列表
7. 测试「生成本地规则标签」和「导出 Markdown」功能

## 使用说明

### 提取收藏

1. 在小红书收藏页面，确保收藏内容已加载
2. 点击插件图标，然后点击「提取当前页面收藏」
3. 插件会自动提取页面上可见的收藏卡片信息
4. 重复内容会自动去重（基于笔记 ID）

### 生成标签

1. 提取收藏后，点击「生成本地规则标签」
2. 插件会根据标题中的关键词自动生成标签
3. 支持的标签类别：美食、旅行、穿搭、美妆、家居、数码、健身、读书、摄影、职场

### 搜索和过滤

- 在搜索框中输入关键词可以搜索标题或作者
- 使用标签下拉框可以按标签过滤收藏

### 导出 Markdown

1. 点击「导出 Markdown」按钮
2. 浏览器会下载一个 `.md` 文件
3. 可以直接复制内容到 Obsidian 或其他笔记应用

## 权限说明

- `activeTab` - 临时访问当前活动标签页
- `storage` - 本地存储收藏数据
- `host_permissions` - 仅访问小红书网站

## 注意事项

1. 本插件只处理当前页面已加载的内容，不会自动滚动加载更多
2. 需要登录小红书账号才能访问收藏页面
3. 不会下载视频或图片，仅保存链接
4. 数据存储在浏览器本地，清除浏览器数据会丢失收藏

## 开发

如果你想修改或扩展功能：

1. 修改 `popup.js` 中的 `TAG_RULES` 可以添加更多标签规则
2. 修改 `content.js` 中的选择器可以适配页面结构变化
3. 修改 `style.css` 可以自定义界面样式
