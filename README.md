# Memora智能收藏夹——收藏即记忆

一个 Chrome/Edge 浏览器插件，帮助你提取、管理和导出抖音/小红书收藏内容。

## 

1. **提取收藏** - 从抖音/小红书当前页面提取可见的收藏卡片信息
2. **智能标签** - 调用千问3.6-plus接口智能生成标签
3. **收藏夹管理** - 支持以收藏夹形式归类内容，收藏夹+标签形式高效检索
4. **收藏列表导出** - 导出收藏内容为 Markdown 格式，适合复制到 Obsidian

## 食用方式

1. 克隆本仓库或下载ZIP到本地
2. 打开你的 Chrome/Edge 浏览器
3. 访问 `chrome://extensions/`或`edge://extensions/`
4. 开启「开发者模式」
5. 点击「加载已解压的扩展程序」
6. 选择你将本项目下载到的本地目录

### 3. 测试功能

1. 打开抖音/小红书网站并登录
2. 进入个人主页，
3. **刷新一次页面**（确保扩展在页面加载前注入，才能读到首屏数据）
4. 确保页面上有可见的收藏卡片（可先向下滚动加载几屏）
4. 点击浏览器工具栏中的插件图标
5. 点击「提取当前页面收藏」按钮
6. 查看提取的收藏内容列表
7. 测试「生成本地规则标签」和「导出 Markdown」功能

## 使用说明

### 提取收藏

1. 在小红书收藏页面，确保收藏内容已加载
2. 点击插件图标，然后点击「提取当前页面收藏」
3. 插件会自动提取页面上可见的收藏卡片信息
4. 重复内容会自动去重（基于笔记 ID）

### 生成标签

1. 提取收藏后，点击「生成本地规则标签」
2. 插件会根据标题中的关键词自动生成标签
3. 支持的标签类别：美食、旅行、穿搭、美妆、家居、数码、健身、读书、摄影、职场

### 搜索和过滤

- 在搜索框中输入关键词可以搜索标题或作者
- 使用标签下拉框可以按标签过滤收藏

### 导出 Markdown

1. 点击「导出 Markdown」按钮
2. 浏览器会下载一个 `.md` 文件
3. 可以直接复制内容到 Obsidian 或其他笔记应用

## 技术实现

- **Manifest V3** - 使用最新的 Chrome 扩展规范
- **原生 JavaScript** - 不依赖任何框架
- **chrome.storage.local** - 本地存储收藏数据
- **content script** - 从页面 DOM、SSR 状态（`__INITIAL_STATE__`）和收藏分页 API 提取信息

## 权限说明

- `activeTab` - 临时访问当前活动标签页
- `storage` - 本地存储收藏数据
- `host_permissions` - 仅访问小红书网站

## 注意事项

1. 本插件只处理当前页面已加载的内容，不会自动滚动加载更多
2. 需要登录小红书账号才能访问收藏页面
3. 不会下载视频或图片，仅保存链接
4. 数据存储在浏览器本地，清除浏览器数据会丢失收藏

## 开发

如果你想修改或扩展功能：

1. 修改 `popup.js` 中的 `TAG_RULES` 可以添加更多标签规则
2. 修改 `content.js` 中的选择器可以适配页面结构变化
3. 修改 `style.css` 可以自定义界面样式
